# 量潮课堂大数据微专业商业计划

## 商业模式

### 客户细分

企业计划服务并连接的多边群体包括：

1. **高校**：尤其是希望开展校企合作、提升学生就业质量的院校。
2. **学生**：主要面向双一流以外、对就业感到焦虑的在校生。
3. **企业**：希望招聘到即用性强、稳定性高的员工的企业，或需要对现有员工进行体系化培训的企业。

### 价值主张

1. **对学生**：通过模拟真实企业环境，提升学生的主观能动性、自主探索能力，帮助学生提前了解企业需求和职场环境，增强就业信心。
2. **对企业与学员对接**：将培养后的优秀学生直接推荐至合作企业，降低双方搜寻与匹配成本，提升就业率与企业用人满意度。
3. **对高校**：提供合作课程与配套材料，协助高校申报奖项、提升政绩。
4. **对企业培训**：优化企业现有培养体系，提升培训效率与质量。

### 渠道通路

- **学员触达**：小红书、微信公众号、校园大使、企业微信社群
- **高校触达**：校内推荐、线上招标平台
- **企业触达**：企业间口碑推荐、互联网品牌传播
- **服务交付**：学员/高校通过飞书进行线上课程授课；高校额外交付通过企业微信提供课程与合作材料；企业交付通过人才双向匹配平台及飞书企业培训课

### 客户关系

- **学员端**：运营企业微信社群，通过新媒体内容保持活跃，激发购买意向
- **企业端**：建立专属微信咨询群，提供一对一服务
- **高校端**：参照企业端模式进行维护
- **代表端（合作方）**：创建多方代表社群，收集反馈并同步优化进展，提升合作认同感，促进复购与推荐

### 收入来源

**现阶段**：
- 2G：向高校收取校企合作课程开发费
- 2C：面向学员收取课程费用

**未来规划**：以企业端收入为核心，C 端与高校端主要作为信用背书。
- 企业：培养方案优化费、人才推荐费、批量人才培养优惠套餐
- 高校：为建立背书可减免合作费用
- C 端：标准产品付费

### 核心资源

1. 浙江理工大学的高校背书
2. 独创的"议事制"教学模式与课程体系
3. 丰富的初创企业社交资源网络
4. 关键数据与人才网络：学员学习数据、企业招聘偏好与反馈库
5. 核心团队：具备产业经验的研发与教学团队、课堂 BU、新媒体与私域运营团队

### 关键活动

- 课程研发与迭代
- 新媒体获客与内容运营
- 校企合作拓展
- 企业合作签约与维护
- 线上线下授课交付

### 重要伙伴

1. **浙江理工大学计算机系**：课程共建
2. **校园大使与学子公司社团**：招生推广
3. **学生所在高校**：课程共建
4. **量潮科技其他 BU**：交叉销售

### 成本结构

- 团队人力成本
- 线上授课工具与平台费用
- 新媒体内容制作与投放成本
- 线下活动与差旅费用
