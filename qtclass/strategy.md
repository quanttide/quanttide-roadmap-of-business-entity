# 量潮课堂商业计划

## 商业模式

### 客户细分

企业计划服务并连接的多边群体包括：

1. **高校**：尤其是希望开展校企合作、提升学生就业质量的院校。
2. **学生**：主要面向双一流以外、对就业感到焦虑的在校生。
3. **企业**：希望招聘到即用性强、稳定性高的员工的企业，或需要对现有员工进行体系化培训的企业。

### 价值主张

量潮课堂的核心竞争力源于其构建了一个以真实商业实践为驱动、能够自我演进的教育生态系统。该体系的核心优势在于将企业的即时性知识资产无缝转化为教学内容，确保了课程内容始终与前沿的市场需求保持同步。

不同于传统校企合作的浅层对接，量潮通过其独创的"议事型课堂"和分层实训机制，将教学过程设计为一个动态的人才筛选与培养管道。这一管道不仅低成本、高效率地为其母公司输送了高度契合的预备人才，形成了稳定的人才供应链，更关键的是，它创造了一个业务与教学相互滋养的飞轮：教学实践催生了数据工程、创新管理等原创知识体系，而这些体系又反过来增强了母公司的商业竞争力。

更深层次的竞争力在于其底层哲学。量潮课堂坦然接纳并系统化地管理"经验主义"领域固有的不确定性与试错成本，其教学目标并非传递绝对正确的静态知识，而是培养学生具备在复杂真实环境中进行批判性思考和自我修正的元能力。这种以"治理混乱"为核心的教育理念，使其能够灵活适应"草台班子"式的小微企业现实需求，从而在解决产教脱节这一根本问题上，构建了难以复制的模式壁垒和持续创新的内在动力。

对各方具体的价值主张：

- **对学生**：通过模拟真实企业环境，提升学生的主观能动性、自主探索能力，帮助学生提前了解企业需求和职场环境，增强就业信心。
- **对企业与学员对接**：将培养后的优秀学生直接推荐至合作企业，降低双方搜寻与匹配成本，提升就业率与企业用人满意度。
- **对高校**：提供合作课程与配套材料，协助高校申报奖项、提升政绩。
- **对企业培训**：优化企业现有培养体系，提升培训效率与质量。

### 渠道通路

- **学员触达**：小红书、微信公众号、校园大使、企业微信社群
- **高校触达**：校内推荐、线上招标平台
- **企业触达**：企业间口碑推荐、互联网品牌传播
- **服务交付**：学员/高校通过飞书进行线上课程授课；高校额外交付通过企业微信提供课程与合作材料；企业交付通过人才双向匹配平台及飞书企业培训课

### 客户关系

- **学员端**：运营企业微信社群，通过新媒体内容保持活跃，激发购买意向
- **企业端**：建立专属微信咨询群，提供一对一服务
- **高校端**：参照企业端模式进行维护
- **代表端（合作方）**：创建多方代表社群，收集反馈并同步优化进展，提升合作认同感，促进复购与推荐

### 收入来源

**现阶段**：
- 2G：向高校收取校企合作课程开发费
- 2C：面向学员收取课程费用

**未来规划**：以企业端收入为核心，C 端与高校端主要作为信用背书。
- 企业：培养方案优化费、人才推荐费、批量人才培养优惠套餐
- 高校：为建立背书可减免合作费用
- C 端：标准产品付费

### 核心资源

1. 浙江理工大学的高校背书
2. 独创的"议事制"教学模式与课程体系
3. 丰富的初创企业社交资源网络
4. 关键数据与人才网络：学员学习数据、企业招聘偏好与反馈库
5. 核心团队：具备产业经验的研发与教学团队、课堂 BU、新媒体与私域运营团队

### 关键活动

量潮课堂与浙江理工大学计算机系共建的大数据微专业正在实施中，同时也正在启动社会招生。核心活动包括：

- 课程研发与迭代
- 新媒体获客与内容运营
- 校企合作拓展
- 企业合作签约与维护
- 线上线下授课交付

### 重要伙伴

1. **浙江理工大学计算机系**：课程共建
2. **校园大使与学子公司社团**：招生推广
3. **学生所在高校**：课程共建
4. **量潮科技其他 BU**：交叉销售

### 成本结构

- 团队人力成本
- 线上授课工具与平台费用
- 新媒体内容制作与投放成本
- 线下活动与差旅费用
